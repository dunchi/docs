저희 팀이 8월 29일 다음 전사회의까지 계획하는 핵심 목표는, '실제' 서비스 환경에서 작동하는 모습을 시연하는 것입니다.

개요:

  • a. 코어 기능 개발 (업장 추가, 직원 추가, 회원가입, 로그인 등)
  • b. 개발 후 배포 과정 자동화
  • c. 테스트가 아닌 실제 데이터와 이용 가능한 회원가입, 로그인, 근태 등록, 근태 조회 페이지 시연
  • d. 대시보드 페이지 UI 기획

상세:

  1. 비서 플랫폼의 중심인 코어 시스템의 데이터베이스 설계 및 구축은 이미 완료되었습니다.
  2. 배포 자동화와 민감 정보 암호화는 마무리 단계에 있습니다.
  3. 로그인, 회원가입 등은 이미 개발되었으나, 테스트성이 아닌 배포된 서버에서도 정상적으로 동작하는지 검토 중입니다.
  4. 이 검토가 끝나면, 업장 추가, 직원 추가 등과 같은 실제 코어 기능을 개발하기 시작합니다.
  5. 코어 기능이 준비되면, 한 컴퓨터에서 돌아가는 것이 아니라 여러 가상 컴퓨터에서 돌아가기 때문에, 각 서버에서 각 기능들이 서로 연계할 수 있도록 하는 작업을 진행합니다.
  6. 위까지가 백엔드 개발이고, 이 작업이 마무리되면 프론트엔드에서 원석님이 작업을 시작합니다.
  7. 새로 추가된 기능들에 따라 그리드 디자인도 변경하고, 내부 테스트 환경이 아닌 사내 임직원분들도 사용할 수 있도록 실제 데이터베이스와 연결하는 작업이 필요합니다.
  8. 이후 일정이 괜찮다면, 백엔드에서는 계약 기능 개발을 시작하고, 프론트엔드에서는 대시보드나 앞으로 추가될 페이지의 UI 기획을 진행할 예정입니다.
