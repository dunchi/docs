📋개요

개발 및 시스템 관리 과정에서 자주 사용되는 유용한 명령어들을 정리한 문서입니다. 각 명령어는 카테고리별로 분류되어 있으며, 코드 옆의 복사 버튼을 클릭하여 간편하게 복사할 수 있습니다.

🐧프로세스 관리

Linux특정 포트에서 실행중인 프로세스 Kill

지정된 포트에서 실행 중인 모든 프로세스를 강제로 종료합니다. 포트 충돌 해결에 유용합니다.

lsof -ti:7081 | xargs kill -9

🐳Docker 관리

Docker특정 노드의 현재 IP 확인

Docker Swarm 환경에서 특정 노드의 IP 주소를 확인합니다.

docker node inspect 7hr21ku7xtha3cdffxddfh7zp --format '{{.Status.Addr}}'

Docker전체 노드의 현재 IP 확인

Docker Swarm 클러스터의 모든 노드와 해당 IP 주소를 테이블 형태로 출력합니다.

echo "노드명 IP 주소" && echo "==================================================" && for node_id in $(docker node ls --format "{{.ID}}"); do hostname=$(docker node inspect $node_id --format '{{.Description.Hostname}}'); ip=$(docker node inspect $node_id --format '{{.Status.Addr}}'); printf "%-40s %s\n" "$hostname" "$ip"; done

Docker서비스의 실행중인 노드 확인

특정 Docker 서비스가 어느 노드에서 실행되고 있는지 확인합니다.

docker service ps Development_stack_biseo-bff

Docker전체 서비스의 실행중인 노드 확인

Docker Swarm 환경에서 모든 서비스가 어느 노드에서 실행되고 있는지 한 번에 확인합니다.

docker service ls -q | xargs -I {} docker service ps {}

🍎macOS 관리

macOS맥북 IP 주소 확인

macOS에서 현재 사용 중인 네트워크 인터페이스의 IP 주소를 확인합니다.

ifconfig | grep "inet " | grep -v 127.0.0.1 | awk '{print $2}' | head -1
